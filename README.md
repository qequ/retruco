# Retruco
A virtual machine and interpreter for the [TIMBA](http://dirinfo.unsl.edu.ar/servicios/abm/assets/uploads/materiales/ddafb-timba-2018.pdf) language.

# Prerequisites

1. Python3
2. Tkinter

To install Tkinter;
```
pip install tk
```

# Usage 

```
python retruco.py
```
